# vue-swipeable-tab

A swipeable tab Vue component, Compatible with Vue 2.x.  
This component works only on mobile.

## Install

``` sh
npm install vue-swipeable-tab
```

``` js
import SwipeableTab from 'vue-swipeable-tab'

Vue.component('my-component', {
    components: {
        SwipeableTab
    }
});
```
